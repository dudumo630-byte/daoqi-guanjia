## 一、录屏前的准备

### 1.1 准备测试数据

在 `data.json` 里准备 8-12 个真实物品，覆盖：

| 类型 | 数量 | 例子 |
|---|---|---|
| 即将过期（3 天内） | 2-3 个 | 牛奶、酸奶 |
| 已过期 | 1-2 个 | 上周的面包 |
| 中期（30 天内） | 3-4 个 | 鸡蛋、酱料 |
| 长期（90 天+） | 2-3 个 | 化妆品、保健品 |
| 证件类 | 1 个 | 身份证（演示多场景） |

### 1.2 配置好 API Key

- 在「设置」里填入 DeepSeek API Key
- 填入城市（如"北京"）
- 填入用户记忆（如"家有 3 岁宝宝，注重食品安全"）

### 1.3 关闭干扰

- 关闭其他应用通知（微信/邮件）
- 桌面背景用纯色或简洁壁纸
- 准备一张商品图片用于演示 AI 识别

## 二、60 秒分镜脚本

### 镜头 1：开场（0-5 秒）

**画面**：到期管家启动 → 启动弹窗提醒
**字幕**："AI 驱动的家庭物品保质期智能管家"
**画外音**（可选）："中国家庭每年因食品过期浪费 1500 元，我做了个 AI 工具来解决这个问题"

### 镜头 2：主界面 + 列表（5-15 秒）

**画面**：主界面 → 滚动展示分类分组 → 红色角标显示告警
**重点**：分组清晰、状态可视、信息密集

### 镜头 3：AI 图片识别（15-25 秒）⭐

**画面**：点"添加商品" → 选图片 → 点"🔍 AI 识别图片" → 等待 2 秒 → 自动填充表单
**字幕**："AI 多模态识别，一拍即录"
**重点**：从图片到结构化数据，2 秒完成

### 镜头 4：AI 洞察（25-40 秒）⭐

**画面**：点"AI 洞察"按钮 → 小乌龟思考动画 → 4 维度报告渲染
**字幕**："4 维度个性化报告：本周关注 / 处理建议 / 趋势 / 重要提醒"
**重点**：
- 小乌龟动画（差异化体验）
- 报告结构清晰
- 建议具体可操作（不是泛泛而谈）

### 镜头 5：AI 对话（40-55 秒）

**画面**：右侧 AI 助手 → 点建议问题"我这周该先吃什么？" → AI 给出基于物品清单的具体建议
**字幕**："多轮对话，城市感知，上下文记忆"

### 镜头 6：结尾（55-60 秒）

**画面**：主界面全景 → 渐隐到 Logo
**字幕**：
```
到期管家 v1.0
独立产品 + 开发 | 3069 行 Python | 4 个 AI 模块
github.com/dudumo630-byte/daoqi-guanjia
```

## 四、转 GIF（嵌入 README）

### 4.1 用 ffmpeg（命令行）

```bash
# 安装 ffmpeg
brew install ffmpeg

# MOV → GIF（按分镜切成多个 GIF）
ffmpeg -i 到期管家demo.mov -ss 0 -t 10 -vf "fps=10,scale=600:-1" demo_1_intro.gif
ffmpeg -i 到期管家demo.mov -ss 25 -t 15 -vf "fps=10,scale=600:-1" demo_3_insight.gif
```

参数说明：
- `-ss`：开始时间（秒）
- `-t`：持续时间
- `fps=10`：10 帧/秒（够用）
- `scale=600:-1`：宽度 600px，高度按比例

### 4.2 在线工具（无脑版）

- [ezgif.com](https://ezgif.com/video-to-gif)：上传 → 设置尺寸 → 下载
- [giphy.com](https://giphy.com/create/gifmaker)：直接生成可分享链接

### 4.3 命名建议

放在 `screenshots/` 目录：

```
screenshots/
```

## 六、录制后的检查清单

录完后**自己先看一遍**，问 5 个问题：

1. ❓ 第一秒能不能看出"这是个 AI 产品"
2. ❓ AI 图片识别的过程是否清楚（图片→JSON）
3. ❓ AI 洞察报告是否展示了"4 维度结构"
4. ❓ 小乌龟动画是否出现（差异化记忆点）
5. ❓ 60 秒内是否展示了所有核心能力

如果其中有 1 项答"否"，重录。

## 七、常见问题

### Q1：录屏时鼠标移动太快怎么办？
A：刻意放慢。每点一个按钮后停 0.5 秒。AI 响应时（2-5 秒）正好给小乌龟动画时间。

### Q2：API 响应慢怎么办？
A：提前 1 小时跑过一次，DeepSeek 缓存热了。或者剪辑时把等待时间压缩。

### Q3：录屏文件太大？
A：用 ffmpeg 压缩：
```bash
ffmpeg -i input.mov -crf 28 -preset slow output.mp4
```

### Q4：要不要加画外音？
A：第一版**不要**。先做无声字幕版（HR 默认静音看）。后续可以加。
